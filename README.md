\# Telecom Churn Prediction API



\## 🚀 Project Overview

This project predicts customer churn using a Machine Learning model and serves predictions using FastAPI.



\## 🛠 Tech Stack

\- Python

\- FastAPI

\- Scikit-learn

\- Docker



\## 📂 Project Structure

\- app.py → FastAPI application

\- train.py → Model training

\- Dockerfile → Containerization

\### Run locally:
